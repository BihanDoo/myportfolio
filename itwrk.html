    Windows Applications

    Run as Invoker tool

    A tool to run applications with elevated privileges, bypassing User Account Control (UAC) prompts.
    This is a simple tool, which can run .exe files as "not administrator"
    Think about when you want to open a malware-infected executable file, and it automatically asks to run it as administrator, and you are afraid to click "yes"
    then this tool can bypass it
    some applications require you to install the application for all users, which is the reason it asks for the administrator permission
    Can be used to get away with those annoying "run as administrator" pop up when running exe files.
    The tool is open source, and the code is simple. User interface is made with CustomTkinter UI
    You can try the tool on Github

    Source code: https://github.com/BihanDoo/RunAsInvokerTool
